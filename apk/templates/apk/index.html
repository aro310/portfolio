Projets & Réalisations

GED – Gestion Électronique de Documents

Projet académique de l’ESTI visant la conception d’un système de gestion électronique de documents intelligent. La plateforme permet l’organisation, le stockage et la recherche avancée des fichiers universitaires grâce à des fonctionnalités d’ intelligence artificielle telles que OCR, Face-Id, chatbot vocal et recherche intelligente. L’application repose sur une architecture robuste avec Django, offrant une interface moderne et une logique backend fiable.

LOCAR – Application de location

Application web dédiée à la gestion de location de véhicules, intégrant la réservation automatisée, la gestion des clients, la facturation et le suivi des contrats en temps réel. Le projet met l’accent sur la simplicité d’utilisation et l’efficacité, avec un backend développé en Flask et une structure pensée pour l’évolutivité.

Voir le code

AVATARO – Projet presonnel avec un Avatar interactif

Projet personnel expérimental centré sur la création d’un avatar interactif intelligent. L’objectif est de permettre à l’avatar de communiquer de manière naturelle via la voix et le texte en s’appuyant sur plusieurs API d’IA, notamment Gemini. L’interface est développée avec React (Vite) tandis que le backend repose sur Flask, offrant une interaction fluide et immersive.
